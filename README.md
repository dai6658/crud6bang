•	Hướng dẫn cài đặt và sử dụng
Download crud6bang-main.zip từ github
Giải nén ra, được 1 foler tên là crud6bang-main, đổi tên thành crud6bang, paste folder crud6bang vào trong folder htdocs
Kiểm tra kết nối: Hệ thống kết nối tới MySQL Server.
Mở Xampp, khởi động apache và mysql
Cấu hình server:
Tên csdl: ktx_management
Server: localhost
Password:
User: root
Port: 3306
Tạo Database: vào  http://localhost/crud6bang/index.php, ấn vào chạy file setup.php
Trong trường hợp không dùng port 3306: Tìm các lệnh $con hoặc $conn ở mỗi file, chỉnh thành port bạn đang sử dụng
